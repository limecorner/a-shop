# a-shop

## 介紹

簡易電商

### 功能

- 表單流程
- 增減商品數量，並查看目前總費用

## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

  ```bash
  npm install
  ```

4. 安裝完畢後，繼續輸入：

  ```bash
  npm run watch
  ```
  
5. 若看見此行訊息則代表順利運行
  ```bash
  webpack 5.68.0 compiled successfully in xxx ms
  ```

6. 在VS Code裡，點index.html檔案，在index.html標籤按右鍵，copy path，把此段網址貼到瀏覽器上


7. 欲暫停使用

  ```bash
  ctrl + c
  ```
